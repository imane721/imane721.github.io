Imane HTML Lab from Day1

We'll be editing a remote file so we can't use a GUI based editor to do our editing. Use a terminal based editor to edit your files. We've used nano before, which you can use. I encourage you to try using emacs or vim which are more powerful/popular editors, but which have a bit of a steep learning curve. You'll want to find and consult a cheat sheet if you choose to use one of them. Use the w3c validator to make sure you don't have any syntax errors. Your HTML only webpage will be very boring looking. Spruce it up by add a CSS stylesheet to make it more interesting. Try experimenting with colors, fonts, borders, etc. Submit (to your main class repo) a text file that contains the URL for your webpage. Name the text file "myWebpageURL.txt".
